# Weather App
### Created by _Maxwell Cady_ and _Ryan Mathisen
#### A simon says game. The user must match a pattern of flashing colors across the screen.

## Setup Instructions
* Clone repo
* npm start
* gulp build
* navigate to index.html

## Technologies Used

* JavaScript
* jQuery
* Gulp
* OpenWeather API
